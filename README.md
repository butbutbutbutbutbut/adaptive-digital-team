# Adaptive Digital Team

This repository is the governance root for the Adaptive Digital Team.

The Persistent Holder control-plane protocol is a
`CANDIDATE_FOR_INDEPENDENT_REVIEW` at
`protocols/PERSISTENT_HOLDER_CONTROL_PLANE.md`. It defines durable routing,
temporary Maker/Checker roles, task-scoped publication leases, and recovery.

Runtime, automation, scheduling, Agent creation, product implementation,
Ready, Merge, and final acceptance remain unimplemented or Human-only.

See `PROJECT_STATE.md` for the current candidate state and next gate.
